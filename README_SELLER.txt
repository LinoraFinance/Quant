Seller setup:
1. Upload this folder to the GitHub repository connected to Netlify.
2. In Netlify enable Identity.
3. Set Registration to Invite only.
4. Enable Git Gateway.
5. Invite the buyer email in Identity → Users.
6. Buyer logs in at /admin.

This version includes a full Decap CMS configuration with global settings, theme settings, image manager, full page HTML editor, service editor, legal editor and advanced raw editing.
