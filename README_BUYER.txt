QUANTHEXA Editable Website — Buyer Guide

Admin URL: /admin
Login: use the email invited through Netlify Identity.

What can be edited:
1. Global Website Settings — company name, contacts, SIREN, header, menu, footer, logo, favicon.
2. Theme / Visual Style — colours, matrix effect, cursor, sounds, custom CSS.
3. Image Manager / All Pictures — logo, favicon, hero background, all solution/service images, plus custom image replacement rules by CSS selector.
4. Full Page Editor — complete HTML/CSS/JS for every page.
5. Service Cards / Prices — service names, prices and descriptions.
6. Legal Text Editor — privacy, refund and terms.
7. Advanced / Raw Config Editor — direct HTML/CSS/JS editing for advanced users.

How to change any picture:
Admin → 03 Image Manager / All Pictures → upload/change the image → Publish → wait for Netlify deploy → refresh website with Ctrl+F5.

How to change any text/layout:
Admin → 04 Full Page Editor → choose page → edit FULL PAGE CONTENT HTML → Publish → wait for deploy.

Important:
After publishing, wait until Netlify deploy is finished. Then open the website and press Ctrl+F5 to clear cache.
